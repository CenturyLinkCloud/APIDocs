This method will log you out of the API. The Logon method must be called again prior to accessing the API again.

## URL

    REST: https://api.ctl.io/REST/Auth/Logout/

## Request

### Attributes

None Required.

### Status Codes

| Status Code | Description |
| --- | --- |
| 0 | Logout was successful |
